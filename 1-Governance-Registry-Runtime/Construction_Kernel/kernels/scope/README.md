# Construction_Scope_Kernel

Canonical source of **scope-domain truth** for the construction-kernel family.

## Role

This repository owns scope-domain truth: scope of work, operations, sequencing, trade responsibility, inspection steps, commissioning steps, closeout, and warranty handoff posture.

## Initial Domain Focus

CSI Division 07 — Building Envelope Systems.

## Ownership Boundaries

This kernel does **not** own:

- Specification truth (see Construction_Specification_Kernel)
- Assembly truth (see Construction_Assembly_Kernel)
- Material truth (see Construction_Material_Kernel)
- Chemistry truth (see Construction_Chemistry_Kernel)
- Reference intelligence (see Construction_Reference_Intelligence)

## Family

Part of the **construction-kernel** family. Registry: ValidKernel_Registry.
