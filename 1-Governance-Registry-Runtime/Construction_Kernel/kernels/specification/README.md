# Construction_Specification_Kernel

Canonical source of **specification-domain truth** for the construction-kernel family.

## Role

This repository owns specification-domain truth: specification documents, sections, source pointers, requirements, prohibitions, allowances, conditions, submittal/testing/warranty requirements, standards references, and revision posture.

## Initial Domain Focus

CSI Division 07 — Building Envelope Systems.

## Ownership Boundaries

This kernel does **not** own:

- Assembly truth (see Construction_Assembly_Kernel)
- Material truth (see Construction_Material_Kernel)
- Chemistry truth (see Construction_Chemistry_Kernel)
- Scope truth (see Construction_Scope_Kernel)
- Reference intelligence (see Construction_Reference_Intelligence)

## Family

Part of the **construction-kernel** family. Registry: ValidKernel_Registry.
