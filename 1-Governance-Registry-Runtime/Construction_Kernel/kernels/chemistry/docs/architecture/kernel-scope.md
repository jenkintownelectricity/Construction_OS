# Chemistry Kernel Scope

## Domain

Chemistry-domain truth for CSI Division 07 — Building Envelope Systems.

## Object Types Governed

| Object Type | Description |
|---|---|
| Chemical System | A formulated chemistry (sealant, adhesive, coating, primer, cleaner, membrane component) |
| Polymer Family | A polymer chemistry class with characteristic behaviors |
| Additive | A chemical additive within a system (plasticizer, stabilizer, filler, catalyst) |
| Cure Mechanism | A cure pathway with environmental dependencies |
| Solvent System | A solvent profile with VOC, flash point, evaporation data |
| Adhesion Rule | A substrate-specific adhesion profile with prep and primer requirements |
| Incompatibility Rule | A pairwise chemical conflict with mechanism and severity |
| Degradation Mechanism | A chemical degradation pathway with rate factors |
| Chemical Hazard Record | A hazard classification from SDS data |

## Division 07 Coverage

Chemistry truth applies across all Division 07 sections where chemical behavior governs performance:
- 07 10 00 — Dampproofing and Waterproofing (bituminous, polyurethane, epoxy coatings)
- 07 20 00 — Thermal Protection (adhesive compatibility with insulation)
- 07 40 00 — Roofing and Siding Panels (sealant chemistry at panel joints)
- 07 50 00 — Membrane Roofing (SBS, APP, EPDM, TPO, PVC chemistry)
- 07 60 00 — Flashing and Sheet Metal (sealant-to-metal adhesion)
- 07 70 00 — Roof and Wall Specialties (sealant and adhesive chemistry)
- 07 80 00 — Fire and Smoke Protection (intumescent chemistry)
- 07 90 00 — Joint Protection (sealant cure, adhesion, compatibility)

## Exclusions

- Physical material properties (Material Kernel)
- Assembly sequences (Assembly Kernel)
- Specification clauses (Specification Kernel)
- Trade scope (Scope Kernel)
- Intelligence synthesis (Reference Intelligence)

## Growth Path

Initial focus is Division 07. Future expansion to other divisions follows the same chemistry-truth model with new taxonomy entries but identical schema structures.
