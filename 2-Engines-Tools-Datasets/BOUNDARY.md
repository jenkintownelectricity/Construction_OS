# Boundary: 2-Engines-Tools-Datasets

## Classification
Ring 2 — Engines, Tools, and Datasets

## Purpose
Contains engine definitions, tool registries, and dataset schemas.
These are the computational and data surfaces of the Construction OS domain.

## Rules
- No code files (.py, .js, .ts, .tsx, .rs) in this folder during Wave 0
- Engine definitions must reference their source kernel
- Tool registries must align with ValidKernel_Registry
- Dataset schemas must conform to ValidKernel_Specs
- No content may override Ring 0 or Ring 1 doctrine

## Boundary Lock
This folder boundary is frozen as of Construction OS v2 genesis.
Modification requires L0 command authority.
