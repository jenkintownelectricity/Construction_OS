# Minimum Truthful Workflow

## A. Current Real Workflow (Evidence-Based)

```
OPERATOR
  ↓
CONDITION SELECTION (manual — pick from canonical_conditions_v2.json, 120 conditions)
  ↓
ASSEMBLY RESOLUTION (manual — read assembly JSON from assemblies/barrett/ or output/barrett_pmma_packet/json/)
  ↓
PARAMETRIC GEOMETRY (generators/pmma/pmma_flash_generator.py — 10 condition functions, reads calibration specimen)
  ↓
SVG GENERATION (direct parametric SVG in output/barrett_pmma_packet/print/ — white background, PRINT_STANDARD)
  ↓
PDF COMPILATION (tools/export_svg_to_pdf.py — cairosvg + pypdf, produces combined 10-page PDF)
  ↓
DXF EXPORT (tools/export_assembly_to_dxf.py — ezdxf, produces layered DXF per condition)
  ↓
CLIENT PACKET (output/barrett_pmma_packet/client/Barrett_PMMA_Detail_Packet_v1.pdf)
```

**Repos involved:** 10-Construction_OS only
**Scripts involved:** 3 Python scripts
**Evidence:** PDF and DXF artifacts generated and pushed to GitHub in this session
