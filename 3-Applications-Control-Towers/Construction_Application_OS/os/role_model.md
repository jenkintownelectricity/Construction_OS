# Role Model — Construction Application OS v0.1

## Roles

| Role | Description | Primary Applications |
|------|------------|---------------------|
| Project Manager | Initiates workflows, reviews outputs, manages project scope | Both |
| Estimator | Consumes spec intelligence for opportunity evaluation and bidding | Spec Intelligence App |
| Detailer | Reviews, annotates, and finalizes shop drawing outputs | Assembly Parser App |
| System | Automated pipeline execution — no human interaction required | Both |

## Role-Application Matrix

| Role | Assembly Parser | Spec Intelligence |
|------|----------------|-------------------|
| Project Manager | Initiate, Review | Initiate, Review |
| Estimator | — | Consume output |
| Detailer | Review, Finalize | — |
| System | Execute pipeline | Execute pipeline |
